## Ключевые выводы (честно)
1. **M0 (flat) ОТВЕРГНУТ overwhelmingly** для обоих MLIP (ΔBIC(M0−M1) = 720 / 1296 ≫ 6).
   → По свободной энергии вдоль CV smooth-min(d_FeH) **есть структура, это НЕ flat/делокализованный профиль.**
   Это НЕ противоречит NS-1: NS-1 о ТОПОЛОГИИ носителя (на каком атоме сидит H — делокализован), NS-3 о
   СВОБОДНОЙ ЭНЕРГИИ вдоль координаты детачмента — ортогональные вопросы (§6 дизайна).
2. **M2 (double-well) сильно бьёт M1** (ΔBIC(M1−M2) = 706 / 1157 ≫ 6, BIC и AICc согласны) — landscape
   мультимодальный над 1.5–3.6 Å (согласуется с многошаговой carrier-sequence chgnet «Fe→S→O→…»).
3. **⚠️ КРИТИЧНО — высоты барьера из полиномов НЕДОСТОВЕРНЫ:** χ²/dof ≫ 1 для ВСЕХ моделей (даже M2: 22 / 21).
   Низкостепенной полином mis-specifies осциллирующий mean-force (F' меняет знак несколько раз на 1.5–3.6 Å:
   −33→+156→−25→+48→+89). Полиномиальные ΔF‡ (MACE M2 1.46 эВ, CHGNet M1 CRB-blow-up 67000 эВ) =
   **артефакты edge-extrema, НЕ цитировать.** **Достоверная высота = JSON трапеция-PMF** (MACE 0.67, CHGNet 0.30 эВ).
   **MDL/BIC отвечает ТОЛЬКО на вопрос ФОРМЫ/идентифицируемости (flat vs structured) — он робастен; на ВЫСОТУ — нет.**
4. **CRB / σ(ΔF‡):** для планирования взята JSON-трапеция σ_dF (= достоверный CRB-прокси), НЕ ill-conditioned
   полиномиальный CRB. По высоте оба идентифицируемы (σ/ΔF‡ = 0.19 mace, 0.04 chgnet < 0.5).
5. **Сколько окон/времени для √Var(ΔF‡)<0.1 эВ:** t_mult = N_mult = (σ_cur/0.1)².
   - **MACE: ~1.70× длины s136-окна** (доминируют недосэмплированные седло-окна id5 τ_int=130, σ_F=51; id6 σ_F=24).
   - **CHGNet: уже достигнут** (σ=0.011 ≪ 0.1) — порог выполнен на текущих окнах.
